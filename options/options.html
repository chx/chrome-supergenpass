Instructions

  1. Store master password on this options page.

    Don’t worry, the password itself is not stored, only a one-way hash is stored.

  2. In a password box, choose your password from the menu by pressing button 1 to 9.

  3. The first time you use a password after opening your browser, the extensions will ask you to unlock it.

    Careful!
    Evil website owners might want to steal your password, so always make sure that the popup says ‘SuperGenPass for Google Chrome™ by Denis extension says’ and not ‘The page at ... says:’!

  4. That’s it! The password field is now filled with your generated password.
